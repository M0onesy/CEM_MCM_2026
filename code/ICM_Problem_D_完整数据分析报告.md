# 第五部分：模型计算过程详解

## 5.1 Model A: 球员价值评估模型

### 理论基础

**Total_Value = α × Performance_Value + β × Commercial_Value**

其中：α = 0.4, β = 0.6 (体现WNBA商业驱动特性)

### 5.1.1 表现价值计算流程

**Step 1: 估算BPM (Box Plus-Minus)**

公式: `BPM_est = 0.86×PPG_per36 + 0.68×RPG_per36 + 0.89×APG_per36 - 0.79×TOPG_per36 - 2.0`

**计算示例：Caitlin Clark**

原始数据:
- PPG = 19.17, RPG = 5.64, APG = 8.43, TOPG = 5.43, MPG = 35.52

Per 36分钟转换:
- PPG_per36 = 19.17 / 35.52 × 36 = **19.43**
- RPG_per36 = 5.64 / 35.52 × 36 = **5.72**
- APG_per36 = 8.43 / 35.52 × 36 = **8.54**
- TOPG_per36 = 5.43 / 35.52 × 36 = **5.50**

BPM计算:
```
BPM = 0.86×19.43 + 0.68×5.72 + 0.89×8.54 - 0.79×5.50 - 2.0
    = 16.71 + 3.89 + 7.60 - 4.35 - 2.0
    = **21.85**
```

**Step 2: 计算WAR**

公式: `WAR = (BPM - Replacement_Level) × MIN / Season_Minutes_Benchmark`

参数: Replacement_Level = -2.0, Season_Minutes_Benchmark = 2400

```
WAR = (21.85 - (-2.0)) × 1492 / 2400
    = 23.85 × 1492 / 2400
    = **14.83**
```

**Step 3: 计算表现价值**

公式: `Performance_Value = WAR × $0.5M`

```
Performance_Value = 14.83 × $0.5M = **$7.42M**
```

**Step 4: 计算商业价值 (收入归因)**

2024赛季收入增量: $34M - $9.1M = $24.9M (+273%)

Clark归因比例: 70%

```
Commercial_Value = $24.9M × 70% × 0.6 = **$15.2M**
```

**Step 5: 计算总价值**

```
Total_Value = 0.4 × $7.42M + 0.6 × $15.2M = **$12.09M**
```
